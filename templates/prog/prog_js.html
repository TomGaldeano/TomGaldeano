  • Variables
    • var x = 5; var y = 6; var z = x + y;
    • let x = 5; let y = 6; let z = x + y;
    • const PI = 3.14;
    • Variables defined with var are hoisted to the top and can be initialized at any time. Meaning: You can use the variable before it is declared
    • Variables defined with const cannot be reassigned.
    • Variables defined with const must be assigned a value when they are declared.
    • Variable names can contain letters, digits, underscores, and dollar signs.
    • Variables must begin with a letter, underscore, or dollar sign.
    • Variable names are case sensitive (y and Y are different variables).
    • Reserved words (like JavaScript keywords) cannot be used as variable names.
    • hyphens are not allowed in names
    • let and const have block scope, can not be redeclared, must be declared before use, do not bind to this and are not hoisted
    • var does not have have to be declared, is hoisted and binds to this
