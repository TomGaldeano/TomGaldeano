  • Variables in Python are created when you assign a value to them. No need to declare the type explicitly.
    x = 5
    y = "Hello"
  • Variable names must start with a letter or underscore, and can contain letters, numbers, and underscores.
    _my_var = 10
    myVar2 = 20
  • Python is case-sensitive: myvar, MyVar, and MYVAR are different variables.
  • You can assign multiple values to multiple variables in one line:
    a, b, c = 1, 2, 3
  • You can assign the same value to multiple variables:
    x = y = z = "Python"
  • To swap values between variables:
    a, b = b, a
  • Use type() to check the type of a variable:
    x = 5
    print(type(x)) # <class 'int'>
  • You can change the type of a variable by assigning a value of a different type:
    x = 5
    x = "Now I'm a string"
  • Variables can be deleted using del:
    x = 10
    del x
  • Constants are usually named in all uppercase, but Python does not enforce constant values:
    PI = 3.14159

Conditionals

  • Python uses if, elif, and else for conditional statements.
    x = 10
    if x > 5:
      print("x is greater than 5")
    elif x == 5:
      print("x is 5")
    else:
      print("x is less than 5")
  • Indentation is required for code blocks inside conditionals.
  • You can use logical operators: and, or, not.
    if x > 5 and x < 20:
      print("x is between 5 and 20")
  • The ternary operator allows for one-line conditionals:
    result = "Yes" if x > 5 else "No"

Loops

  • for loops iterate over sequences (lists, tuples, strings, etc.):
    for i in range(5):
      print(i)
  • while loops repeat as long as a condition is true:
    count = 0
    while count < 5:
      print(count)
      count += 1
  • break exits the loop early, continue skips to the next iteration.
    for i in range(10):
      if i == 5:
        break
      print(i)
  • else can be used with loops and runs if the loop completes without a break:
    for i in range(3):
      print(i)
    else:
      print("Done!")
  • You can loop over lists, strings, dictionaries, etc.:
    for char in "Python":
      print(char)

Functions

  • Functions are defined using def:
    def greet(name):
      print("Hello, " + name)
  • Call a function by its name:
    greet("Tom")
  • Functions can return values using return:
    def add(a, b):
      return a + b
    result = add(2, 3)
  • Default parameter values:
    def greet(name="World"):
      print("Hello, " + name)
  • Use *args for variable number of positional arguments, **kwargs for keyword arguments:
    def demo(*args, **kwargs):
      print(args, kwargs)
  • Functions can be nested and can be assigned to variables.

Classes

  • Classes are defined using class:
    class Person:
      def __init__(self, name):
        self.name = name
      def greet(self):
        print("Hello, " + self.name)
  • Create an object (instance) of a class:
    p = Person("Tom")
    p.greet()
  • __init__ is the constructor method, called when an object is created.
  • Use self to access instance variables and methods.
  • Classes can inherit from other classes:
    class Student(Person):
      def study(self):
        print(self.name + " is studying.")
  • You can override methods in child classes.

String Inbuilt Methods

  • len(s) — Returns the length of the string.
    s = "Python"
    print(len(s)) # 6
  • s.lower() — Converts all characters to lowercase.
    print("HELLO".lower()) # hello
  • s.upper() — Converts all characters to uppercase.
    print("hello".upper()) # HELLO
  • s.strip() — Removes leading and trailing whitespace.
    print(" hello ".strip()) # hello
  • s.replace(old, new) — Replaces all occurrences of old with new.
    print("banana".replace("a", "o")) # bonono
  • s.split(sep) — Splits the string into a list using sep as the separator.
    print("a,b,c".split(",")) # ['a', 'b', 'c']
  • s.find(sub) — Returns the lowest index of sub in the string, or -1 if not found.
    print("hello".find("e")) # 1
  • s.startswith(prefix) — Returns True if the string starts with prefix.
    print("python".startswith("py")) # True
  • s.endswith(suffix) — Returns True if the string ends with suffix.
    print("python".endswith("on")) # True
  • s.isdigit() — Returns True if all characters in the string are digits.
    print("123".isdigit()) # True
  • s.isalpha() — Returns True if all characters in the string are alphabetic.
    print("abc".isalpha()) # True
  • s.join(list) — Joins elements of a list into a string, separated by s.
    print("-".join(["a", "b", "c"])) # a-b-c
