Element selection

  • #id - Selects the element with that id
  • .class - Selects the elements with that class
  • * - Selects all the elements
  • element - selects all those elements
  • element, element, ... - selects all those elements
  • element.class applies only to that element if it has the class
  • element element applies to that element inside another element
  • element > element applies to that element that is a child of another element
  • element + element applies to that element that is immediately preceded by another element
  • element ~ element applies to that element that is preceded by another element
  • [attribute] - Selects elements with a specific attribute
  • [attribute=value] - Selects elements with a specific attribute value
  • [attribute~=value] - Selects elements with a specific attribute value in a space-separated list
  • [attribute|=value] - Selects elements with a specific attribute value in a hyphen-separated list
  • [attribute^=value] - Selects elements with an attribute value starting with a certain value
  • [attribute$=value] - Selects elements with an attribute value ending with a certain value
  • [attribute*=value] - Selects elements with an attribute value containing a certain value
  • element:pseudo-class - Selects elements based on a certain state
  • element::pseudo-element - Selects and styles a part of an element

Text

  • font-family changes the font of the text. Example: font-family: Arial, sans-serif;
  • font-size sets the size of the text. Example: font-size: 16px;
  • font-weight sets the thickness of the text. Example: font-weight: bold;
  • font-style sets the style of the text (normal, italic, oblique). Example: font-style: italic;
  • text-align aligns the text horizontally. Options: left, right, center, justify.
  • text-align-last aligns the last line of text. Options: auto, left, right, center, justify, start, end.
  • color sets the color of the text. Example: color: #333;
  • line-height sets the space between lines of text. Example: line-height: 1.5;
  • letter-spacing sets the space between characters. Example: letter-spacing: 2px;
  • word-spacing sets the space between words. Example: word-spacing: 10px;
  • text-transform controls capitalization. Options: uppercase, lowercase, capitalize.
  • text-decoration adds decoration to text (underline, overline, line-through, none). Example: text-decoration: underline;
  • text-shadow adds shadow to text. Example: text-shadow: 2px 2px 5px #888;
  • vertical-align aligns text vertically (baseline, sub, super, top, bottom, middle, text-top, text-bottom).
  • white-space controls how whitespace is handled. Example: white-space: pre;
  • overflow-wrap and word-break control how words break onto the next line.
  • direction sets the text direction (ltr, rtl).
  • unicode-bidi controls the embedding of bidirectional text.
  • text-indent indents the first line of text. Example: text-indent: 30px;
  • text-overflow handles overflowed text (clip, ellipsis).
  • text-justify sets the justification method (auto, inter-word, inter-character, none).
  • text-orientation sets the orientation of text in vertical writing modes (mixed, upright, sideways).
  • text-rendering provides hints to the browser about how to optimize text rendering (auto, optimizeSpeed, optimizeLegibility, geometricPrecision).
  • hyphens controls hyphenation of text (none, manual, auto).

Colors

  • How to make color border style="border:2px solid Tomato;"
  • rgba(255, 99, 71, 0.2). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)
  • hsl(hue, saturation, lightness)
    • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
    • Saturation is a percentage value. 0% means a shade of gray, and 100% is the full color.
    • Lightness is also a percentage value. 0% is black, and 100% is white.
  • HSLA is the same as with RGBA

Backgrounds

  • background-color changes the background of the page or element
  • background-image url("img_tree.png"). By default, the image is repeated so it covers the entire element.
  • background-repeat property sets if/how a background image will be repeated. They are no-repeat repeat-x repeat-y
  • background-position right top; this sets the image at top right
  • background-attachment property specifies whether the background image should scroll or be fixed options are fixed scroll
  • background-clip specifies whether an element's background, either the color or image, extends underneath its border. Initial Value: border-box .Values: border-box, padding-box, content-box, text.
  • background-origin CSS property determines the background positioning area, that is the position of the origin of an image specified using the background-image CSS property. Initial Value: padding-box. Values: border-box, padding-box, content-box.
  • background-size Specifies the size of the background image(s)

Margins

  • margin-topsets the top margin of an element
  • margin-rightsets the right margin of an element
  • margin-bottomsets the bottom margin of an element
  • margin-leftsets the left margin of an element
  • margin: 25px 50px 75px 100px; this sets the margin in clockwise order starting from top
  • margin: 25px 50px 75px; this sets the top margin to 25px, right and left to 50px, and bottom to 75px
  • margin: 25px 50px; this sets the top and bottom margins to 25px, and the right and left margins to 50px
  • margin sets all the margins of an element (top, right, bottom, left)
  • margin: auto;The element will then take up the specified width, and the remaining space will be split equally between the left and right margins.
  • margin: inheritlet the margin be inherited from the parent element.
  • Top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins. Note: Margin collapse only happens with top and bottom margins! Not left and right margins

Box model

  • box-sizing defines how the width and height of an element are calculated: should they include padding and borders, or not.
    • box-sizing:content-box;This is default. The width and height properties includes only the content (border and padding are not included)
    • box-sizing:border-box;The width and height properties includes content, padding and border
  • max-widthproperty is used to set the maximum width of an element.
  • heightand widthproperties are used to set the height and width of an element. And have the following property values
    • autoDefault. browsert calculates the height and width
    • defined in px, cm, em ..
    • %defines width or height in percent of the containing block
    • initial sets the height or width to its default value
    • inheritinherits values from parent value
  • min-widthproperty used to set the minimum width of an element.
  • max-heightproperty used to set the maximum height of an element.
  • min-heightproperty uded to set the minumum height of an element.
